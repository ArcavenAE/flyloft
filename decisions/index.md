# Architecture Decision Records

## Index

No ADRs yet.

ADRs for flyloft follow the MADR format (see aae-orc orchestrator for
the template). Record a decision when you:

- Choose between alternatives with non-obvious tradeoffs
- Commit to a direction that's expensive to reverse
- Want the reasoning preserved for future readers

Not every design choice needs an ADR. Reserve them for decisions that
would be hard to reconstruct from code review alone.
